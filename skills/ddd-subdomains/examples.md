# Subdomains — 走查示例

> 附加文件，按需读取。

```text
输入：泳道 [预订核心][审批][通知][身份] + 价值主张。

子域分类表：
  | 预订与冲突 | Core | 价值主张"订到=有房"全靠它；竞品做不到细粒度冲突检测 | 自研深耕 |
  | 审批 | Supporting | 业务需要但规则常变、非差异化 | 轻定制（可配置规则）|
  | 通知 | Generic | 邮件/IM 推送人人一样 | 采购/接现有 |
  | 身份 | Generic | 接公司 LDAP（硬约束）| 集成现成 |

核心域声明：«预订与冲突» —— 对应价值主张 1「订到=一定有房」。

投入建议：预订与冲突=高（最强建模最严不变量）；审批=中；通知/身份=低。
```
